INSTALACIÓN SERVIDOR FTP EN UBUNTU SERVER 16.04

Para la instalación del servidor FTP autenticado en Ubuntu Server 16.04 se deben seguir los siguientes pasos:


1. Inicialmente nos aseguramos mediante el comando “sudo su” estar como súper usuario, luego se instala el servidor FTP utilizando el comando “apt-get install vsftpd”.



2. Para permitir la carga y descarga de documentos accedemos a al archivo de configuración de FTP a través de la ruta “/etc/vsftpd.conf” en él se busca la línea de texto “write_enable=YES” y se descomenta



3. Ahora se descarga e instala un cliente FTP, para el caso de este trabajo se usará FileZilla Su sitio de descarga oficial es “https://filezilla-project.org/”



4. La ruta donde se guardan los archivos transferidos mediante FTP es “/home/Ubuntu/”, en nuestra máquina virtual accedemos a ella y ya instalado en cliente FTP iniciamos la conexión ingresando la dirección IP del servidor, el usuario y la contraseña..



Nota:De lado izquierdo se listan los directorios locales, y de lado derecho los directorios del servidor. Para realizar la transferencia solo basta con arrastrar y soltar, Si FTP ha quedado correctamente instalado no habrá ningún error.
